# Nini Yining Lin — Website

This is a  personal academic website hosted on GitHub Pages.

## How to publish
1. Go to your repo **Settings** → **Pages**
2. Under **Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
3. Save, then wait for GitHub to provide the URL.

## Edit content
- Update text in `index.html`
- Update styles in `styles.css`

## Optional
- Add a CV PDF to the repo (e.g., `cv.pdf`) and link it in the Contact section.
- Replace placeholder links (paper link, Scholar, GitHub).
